# GlomeraAI-Diabetic-Nephropathy-Stage-Prediction-APP
DKD Clinical Decision Support System - A multimodel, explainable, and fairness-audited AI system for predicting Diabetic Kidney Disease (DKD) progression across six KDIGO 2022 stages. Built with a stacked ensemble of three sub-models (Random Forest · XGBoost · Logistic Regression) trained on NHANES 2015–2020 cross-sectional data.
